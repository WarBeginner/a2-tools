Все ячейки с ключевым содержимым защищены от изменения.
Что требует внимания:
Лист "Структура" задает порядок и содержание выгрузки. Защищен.
Первая строка на каждом листе данных содержит "внутренние" имена полей. Защищены.
Вторая строка на каждом листе данных содержит "человеческие" имена полей. Можно менять.
Названия листов "человеческие", но при изменении обязательно надо изменить нужное место листа "Структура", иначе таблица не выгрузится. НЕ защищены.
Имеются вложенные таблицы.
  На основном листе в ячейке ставится "ссылка" (По умолчанию - Номер строки в квадратных скобках).
  На листе вложенной таблицы первая колонка содержит такую же ссылку.
  Все строки со "вложенного" листа, содержащие то же значение ссылки (и колонки кроме ссылки),
    образуют вложенную таблицу в качестве содержимого соответствующей ячейки основного листа.
  Содержимое "ссылок" может быть любым текстом.
